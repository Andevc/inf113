elemento iFrame

este elemento permite representar un contexto de navegación anidado, lo que permite al navegador inscrustar un documento WEB con el atributo src. dicho de otra forma permite insertar elementos como: texto, video, imagen, etc. Sus atributos principales son:

a) src:

referencia al documento o URL qu se mostrara en el frame

b) width/heigth:

ancho y alto del marco

c) name:

nombre o identificador del marco respectivamente

d) sandbox:

se utiliza para incorporar medidas o restricciones de seguridad en el frame

e) srcdoc:

Tiene precedencia sobre src, para que el documentop sea mostrado en el freame

f) scrolling:

g) align

h) frameborder:

i) longdesc:

Descripcion larga del frame

j) marginwidth/marginheigth

Ancho y alto del margen izquierdo y derecho respectivamente

k) vspace:

Espacio vertical alrededor del frame

l) style:

Define el estilo del marco mediante CSS
